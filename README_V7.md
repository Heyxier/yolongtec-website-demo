# Yolong 亚隆 - 机械制造企业官网

## 🎨 设计理念

### 设计定位
- **风格**：硬朗、专业、工业感（参考 Milwaukee Tool）
- **主色调**：Pantone 357 C (#006847) - 深绿色
- **点缀色**：Pantone 158 C (#FF3F00) - 橙红色（小面积使用）
- **字体**：Teko（标题）+ Roboto（英文）+ Noto Sans SC（中文）

### 色彩系统
```
主色（Pantone 357 C）
├─ #006847 - 标准色
├─ #005239 - 深色变体
└─ #008F63 - 浅色变体

点缀色（Pantone 158 C）
├─ #FF3F00 - 标准色
└─ #CC2A00 - 深色变体
```

### 设计原则
1. **主色主导** - 深绿色作为品牌主色，大面积使用
2. **点缀克制** - 橙红色仅用于按钮、图标、强调元素（< 5% 面积）
3. **硬朗风格** - 直角、粗线条、大字号标题
4. **专业感** - 简洁布局、清晰信息层级

## 🎭 交互特性

### 动画效果
- **数字滚动** - 统计数据动态增长
- **卡片悬停** - 上浮 + 阴影加深
- **标签页切换** - 平滑过渡
- **滚动淡入** - 元素进入视口时显示
- **导航下划线** - 橙红色从左到右展开

### 响应式设计
- **桌面端** (>1200px): 完整 4 列布局
- **平板端** (768-1200px): 2 列布局
- **移动端** (<768px): 单列布局 + 汉堡菜单

## 🔧 使用说明

### 本地预览
```bash
cd mech-portal
python -m http.server 8080
# 访问 http://localhost:8080
```

### 替换图片
将 `.image-placeholder` 替换为实际产品/新闻图片：
```html
<div class="product-image">
  <img src="your-product-image.jpg" alt="产品名称">
</div>
```

### Logo 使用
- **浅色背景** → 使用 `logo-green.svg`（深绿色文字）
- **深色背景** → 使用 `logo-white.svg`（白色文字）
- 橙色弧线保持不变（Pantone 158 C）
